#######################################INSTRUCTIONS###############################

1- Extraire l'archive zip n'importe où puis se placer dans le dossier extrait. (là où se trouve le fichier Serveur.jar)

2- Lancer le fichier Serveur.jar avec la commande suivante :
			java -jar Serveur.jar
			
	Cette commande va vous permettre de lancer le serveur sur le port 8080

3- Ouvrir un navigateur puis mettre dans la barre d'adresse :
		localhost:8080:/cheminVersDossierExtrait/index.html
		
4- C'est partie ! Vous êtes maintenant connecté au serveur !
